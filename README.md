🎤 Speech to Text Converter (Python)

This is a simple Python project that converts spoken voice into text using the microphone. It uses the SpeechRecognition library and Google’s speech recognition API.

📌 Features

🎙️ Capture audio from microphone
🔊 Convert speech to text
⚡ Fast and simple implementation
🌐 Uses Google Speech Recognition API
🛠️ Requirements

Make sure you have Python 3.x installed.

Install the required libraries:

pip install SpeechRecognition
pip install pipwin
pipwin install pyaudio

📂 Project Structure

speech-to-text/
│── main.py
│── README.md

▶️ How to Run

Open Command Prompt or VS Code terminal
Navigate to your project folder

Run the program:

python main.py

Speak into your microphone when prompted
The recognized text will be displayed
